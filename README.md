# dockerize-vue-app
This is a demo project for dockerizing a Vue.js application.

<img src="screenshots/vue-app.png" alt="vue app" />

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
